## 💻 Sobre el Portafolio

> "La matemática discreta es la columna vertebral de las ciencias de la computación." 

Este repositorio ha sido diseñado como un espacio interactivo para recopilar, estructurar y exponer las evidencias de aprendizaje práctico, autónomo y experimental desarrolladas a lo largo del periodo académico **Marzo – Agosto 2026**. Aquí se documentan desde los fundamentos lógicos hasta el apasionante análisis de estructuras no lineales como grafos y árboles.
